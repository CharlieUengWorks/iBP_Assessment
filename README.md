## HOW TO RUN THIS
1. Install NPM
2. make sure port 3000 and 5000 is free
3. make sure .net 10 runtime is present on the running PC
4. Open a cmd console at the root folder
5. run "npm install"
6. run "npm run build"
7. run "npm run start"
8. start the provided server binary or build a new one with the project [here](https://duckduckgo.com)
9. go to http://localhost:3000
